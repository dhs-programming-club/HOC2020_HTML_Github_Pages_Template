Welcome to the DHS Hour of Code 2020

This year programming club wanted to create a fun and aproachable activity that would allow you to have a tangiable end result, so we decided to teach HTML with Github pages so that you could create an awesome website to showcase your highschool accomplishments.

Getting Started

  • Step 1: Setting up Github

    Start by creating a free github account. After you do this click the link bellow to copy our starting code.

  • Step 2: CODE!

    Follow our guide to code your website into something unique.

  • Step 3: Launch your Website

    Use Github pages to publish your brand new website onto the internet.
